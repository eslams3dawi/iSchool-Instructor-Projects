# iSchool Instructor Projects

This repository contains projects, activities, and educational materials used during my experience as an Instructor at iSchool within the Digital Egypt Marvels Initiative (DEMI).

## Topics Covered

### Programming Fundamentals

* Algorithms and step-by-step problem solving
* Computational thinking
* Teamwork and communication skills

### Game Development with mBlock

* Introduction to Software Development Life Cycle (SDLC)
* Interactive game development
* Sample games:

  * Maze
  * Catch Food
  * Ping Pong

### Artificial Intelligence & Machine Learning

* Introduction to AI concepts
* Training data and pattern recognition
* Machine Learning project using PictoBlox

## Technologies & Tools

* mBlock
* PictoBlox
* Teachable Machine (Machine-Learning Environment)
* Scratch-based Programming

## About

The Digital Egypt Marvels Initiative (DEMI) is presented by the Ministry of Communications and Information Technology (MCIT) and implemented by iSchool.
